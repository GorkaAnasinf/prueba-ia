# Reunión — Nuevo Proyecto Web E-commerce — Distribuidora del Norte — 2026-01-15

## Objetivo
Analizar necesidades del cliente para desarrollo de tienda online B2B. Primera reunión de toma de requisitos.

## Puntos tratados

- Distribuidora del Norte: empresa de distribución alimentaria con 200+ clientes profesionales (hostelería, supermercados).
- Actualmente gestionan pedidos por teléfono, email y WhatsApp. Quieren digitalizar el proceso.
- Catálogo: ~1.800 referencias con variaciones por temporada. Precios diferenciados por cliente/volumen.
- Requisitos clave identificados:
  - Login por cliente con tarifas personalizadas
  - Pedido mínimo configurable por cliente
  - Integración con su ERP (Sage 200) para stock en tiempo real
  - App móvil o web responsive para que los clientes pidan desde el almacén
  - Gestión de rutas de reparto (opcional, fase 2)
- Miren menciona que han mirado Prestashop pero no les convence para B2B.
- Plazo deseado por el cliente: web operativa antes de septiembre 2026.

## Decisiones tomadas

- Se descarta Prestashop. Se propone desarrollo a medida con Next.js + API propia.
- Se hará estudio de viabilidad de integración con Sage 200 antes de presupuestar.
- Pablo Serrano (responsable IT) facilitará acceso a documentación API de Sage.
